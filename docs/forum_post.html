ReaperHaptic

Feel your edits - haptic feedback for Logitech MX Master 4 mouse

What is it?

ReaperHaptic triggers haptic feedback on your Logitech MX Master 4 mouse based on REAPER events. Get tactile "clicks" when items snap together, vibrations when audio clips, subtle feedback when recording starts, and more.

Features

  • Item Snap Detection - haptic "click" when items touch other items, markers, time selection, or playhead
  • Clipping Alert - strong vibration when master track exceeds 0dB
  • Transport Feedback - subtle haptics for record/play start and stop
  • Render Complete - notification when rendering finishes
  • Marker Crossing - light feedback when playhead crosses markers during playback
  • Configuration GUI - toggle individual events, LED indicator, dockable window

Requirements

  • macOS 14+ (Sonoma or later)
  • Logitech MX Master 4 mouse
  • Logi Options+ installed and running
  • REAPER 7.0+
  • mavriq-lua-sockets for REAPER

How it works

REAPER (Lua Script) --OSC/UDP--> ReaperHaptic (Logi Options+ Plugin) --Haptic API--> MX Master 4

The Lua script monitors REAPER events and sends OSC messages to a Logi Options+ plugin, which triggers the appropriate haptic waveforms on your mouse.

Installation

  1. Download ReaperHaptic.lplug4 from GitHub releases - double-click to install
  2. Download reaper_haptic_monitor.lua to your REAPER Scripts folder
  3. Install mavriq-lua-sockets
  4. In REAPER: Actions > Load ReaScript > select the script
  5. (Optional) Add to startup actions for automatic loading
